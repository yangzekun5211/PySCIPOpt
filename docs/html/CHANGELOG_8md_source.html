39 ## 3.0.0 - 2020-04-11
40 ### Added
41 - add Model.getParams that returns a dict mapping all parameter names to their values
42 - add Model.setParams to set multiple parameters at once using a dict
43 - Add Model.from_ptr and Model.to_ptr to interface with SCIP* managed outside of PySCIPOpt
44 - add NULL pointer checks to all Python wrapper classes
45 - add Event.getRow() and Row.name
46 - expose domain changes, bound changes, branching decisions, added constraints for nodes
47 - define Python object identity based on underlying SCIP object pointers, so that e.g. rows and columns can be added to sets, and testing for equality is consistent over time.
48 - add Row.isRemovable and Row.getOrigintype
49 - add Model.applyCutsProbing and Model.propagateProbing
50 - add Model.separateSol
51 - add methods to work with nonlinear rows
52 - adds new "threadsafe" parameter to the PyBendersCopy member function. Also, the "threadsafe" parameter can be passed
53  when creating a Model instance
54 - adds the boolean return options of "infeasible" and "auxviol" to the Benders.benderspresubsolve function. "infeasible"
55  indicates that the input solution induces an infeasible instance of at least one Benders' subproblems. "auxviol"
56  indicates that the objective value of at least on Benders' subproblem is greater than the auxiliary variable value.
57 - adds chgVarUbProbing and chgVarLbProbing to change a variables upper or lower bound during probing mode.
58 
59 ### Changed
60 - Node.getParent() returns None if the node has no parent
61 - remove Python 2.7 support
62 - fix documentation errors
63 - setupBendersSubproblem now requires a checktype input. This input indicates the reason for solving the Benders'
64  subproblems, either enforcing the LP, relaxation or pseudo solution (LP, RELAX or PSEUDO) or checking a candidate
65  primal feasible solution (CHECK).
